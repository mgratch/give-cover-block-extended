This plugin extends the Cover and Image blocks to dynamically use GiveWP P2P Campaign Images and Logos.

== Description ==

After setting up your GiveWP P2P Campaigns, you can use the Cover and Image blocks to dynamically display the Campaign's image and logo. This means that you can create a template for your P2P Campaigns and have the images automatically update when you select a different Campaign or if you change out your logo or image.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/give-cover-block-extended` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
